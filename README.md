# Cainos Basic Importer

This repository now contains a Godot 4.6+ addon that imports the **Pixel Art Top Down - Basic** pack into Godot-native authoring assets.

Current project entrypoint:
- [project.godot](/home/parallels/src/github.com/agentc1/pixel-monsters/godot_project/project.godot)

Main addon files:
- [plugin.cfg](/home/parallels/src/github.com/agentc1/pixel-monsters/godot_project/addons/cainos_basic_importer/plugin.cfg)
- [plugin.gd](/home/parallels/src/github.com/agentc1/pixel-monsters/godot_project/addons/cainos_basic_importer/plugin.gd)
- [basic_pack_importer.gd](/home/parallels/src/github.com/agentc1/pixel-monsters/godot_project/addons/cainos_basic_importer/basic_pack_importer.gd)
- [importer_dock.gd](/home/parallels/src/github.com/agentc1/pixel-monsters/godot_project/addons/cainos_basic_importer/importer_dock.gd)

Use [Quickstart.md](/home/parallels/src/github.com/agentc1/pixel-monsters/godot_project/docs/Quickstart.md) for the beginner workflow and [KnownDifferences.md](/home/parallels/src/github.com/agentc1/pixel-monsters/godot_project/docs/KnownDifferences.md) for the v1 compatibility boundary.
